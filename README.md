# Crossword Puzzle Generator

___
> ## Currently, in development
___

## About
`pycrossword-generator` is a powerful Python library and CLI tool designed for creating custom crossword puzzles. 

___

## Installation
```bash
pip install pycrossword-generator
```
From source:
```bash
make install
```
or
```bash
pip install .
```
